# How to Use the 5 Tiers / Cách Dùng 5 Tầng

### How to Use the 5 Tiers / Cách Dùng 5 Tầng

1. **Tag your raw notes**: When you write in a Taichi journal, highlight or color-code by tier. | Khi viết nhật ký, tô màu theo tầng. |
| 2. **Tag what you re-read**: When you revisit a chapter, mark the tier you need to focus on most. | Khi đọc lại chương, đánh dấu tầng bạn cần tập trung nhất. |
| 3. **Tag your daily practice**: 5 min stand = 🔴, 20 min form = 🟡, breathing = 🔴+🟢, etc. | Gắn tầng cho tập hằng ngày: 5p trụ = 🔴, 20p hình = 🟡, thở = 🔴+🟢, v.v. |
| 4. **Tag what you teach others**: When you teach, focus on Tier 1 (foundation) for beginners, Tier 4-5 for advanced. | Gắn tầng khi dạy: Tầng 1 cho người mới, Tầng 4-5 cho nâng cao. |
| 5. **Use the color as a reminder**: When you see 🔴, think "foundation." When you see 🔵, think "long-term life practice." | Dùng màu làm nhắc: 🔴 = nền tảng, 🔵 = thực hành đời sống dài hạn. |
